# Tech Challenge 5 — Classificação de Candidatos

Este projeto foi desenvolvido como parte do desafio Tech Challenge 5, com foco em **análise e clusterização de perfis de candidatos** para identificação de padrões e apoio à tomada de decisão em processos seletivos.
